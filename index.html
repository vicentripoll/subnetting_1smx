ℹ️ Cómo usar

🧮 Calculadora

Ingresa los parámetros de la red (dirección, máscara CIDR y número de subredes) para visualizar todos los cálculos paso a paso. Ideal para verificar tus cálculos manuales.

📚 Ejercicios

Completa los campos solicitados sin usar la calculadora. Recibe retroalimentación inmediata en verde/rojo y acceso a pistas después de 3 respuestas correctas.

📈 Progresivo

Los ejercicios aumentan en dificultad. Comienza con redes Clase C, sigue con Clase B, y termina con Clase A. Cada intento tiene valores diferentes para mayor aprendizaje.

🎓 Concepto

FLSM (Fixed Length Subnet Mask): Todos los subnets tienen el mismo tamaño. Se calcula mediante 2^n ≥ subredes requeridas para encontrar los bits de subred necesarios.
